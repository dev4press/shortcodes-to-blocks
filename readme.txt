== Description ==
Example plugin for adding Block Editor blocks to replace shortcodes, and add settings for shortcodes attributes.

More information:
* Project on GitHub.com: [ShortCodes to Blocks](https://github.com/dev4press/shortcodes-to-blocks)
* Tutorial on Millan.dev: [Convert shortcodes into blocks](https://millan.dev/2021/04/30/convert-shortcodes-into-blocks/)

== Installation ==
= General Requirements =
* PHP: 7.0 or newer
* WordPress: 5.7 or newer

= PHP Notice =
* The plugin doesn't work with PHP 5.6 or older versions.

= WordPress Notice =
* The plugin doesn't work with WordPress 5.6 or older versions.

= Basic Installation =
* Plugin folder in the WordPress plugins folder should be `archivespress`.
* Upload `archivespress` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* Plugin doesn't have any settings and it doesn't add any panels to WordPress administration.

== Frequently Asked Questions ==
= How can I add blocks to my posts? =
All three blocks registered by this plugin are listed under 'ShortcodesBlocks' block category.

= Where can I configure the plugin? =
The plugin doesn't have any global settings or configuration panels.
